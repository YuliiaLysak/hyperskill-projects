The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

> <host>127.0.0.1</host>
{"host":"127.0.0.1"}


Example 2:

> {"jdk" : "1.8.9"}
<jdk>1.8.9</jdk>


Example 3:

> <success/>
{"success": null}


Example 4:

> {"storage" : null}
<storage/>
